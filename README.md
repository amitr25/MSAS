# MSAS - Mini Secure Authetication System using Cryptography

A comprehensive authentication and authorization demonstration platform built with modern web technologies. This interactive application showcases various security mechanisms including user registration, multi-factor authentication (MFA), role-based access control (RBAC), JWT token management, and end-to-end encryption.

## 🌟 Features

### Authentication & Security
- **User Registration** - Secure account creation with validation
- **Login System** - Credential-based authentication with session management
- **Multi-Factor Authentication (MFA)** - Additional security layer with OTP verification
- **JWT Token Management** - Secure token generation, validation, and refresh mechanisms
- **End-to-End Encryption** - Data protection using modern cryptographic standards

### Authorization & Access Control
- **Role-Based Access Control (RBAC)** - Fine-grained permission management
- **Activity Logs** - Comprehensive audit trail for security monitoring
- **Session Management** - Secure session handling with automatic timeout

### User Experience
- **Interactive Workflow Stepper** - Visual guide through authentication processes
- **Terminal-Style Interface** - Tech-focused, developer-friendly UI
- **Responsive Design** - Seamless experience across all devices
- **Dark/Light Mode** - Theme customization for user comfort

## 🛠️ Tech Stack

**Frontend Framework:**
- React 18.3 with React Router for navigation
- TypeScript for type-safe development
- Vite for lightning-fast builds

**UI Components:**
- shadcn/ui - High-quality, accessible component library
- Radix UI - Unstyled, accessible component primitives
- Tailwind CSS - Utility-first CSS framework
- Lucide React - Beautiful icon library

**State Management:**
- Zustand - Lightweight state management
- TanStack Query - Powerful data synchronization

**Form Handling:**
- React Hook Form - Performant form validation
- Zod - TypeScript-first schema validation

**Testing:**
- Vitest - Fast unit testing framework
- Testing Library - User-centric testing utilities
